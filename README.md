# binary-hygrometer
A humidity reader that outputs values in binary using LEDs

## Installation

Install DHT and Blinkt! depdencies

**Binked!**

```bash
curl https://get.pimoroni.com/blinkt | bash
```

**DHT**

```bash
pip3 install adafruit-circuitpython-dht
sudo apt-get install libgpiod2
```
